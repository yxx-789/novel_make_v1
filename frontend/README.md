# 小说创作平台 - Streamlit 前端

基于 Streamlit 构建的多页面 AI 小说创作平台前端界面。

## 📁 项目结构

```
frontend/
├── app.py              # 主入口文件
├── pages/              # 多页面目录
│   ├── 1_🏠_首页.py
│   ├── 2_✍️_小说创作.py
│   ├── 3_📚_小说管理.py
│   ├── 4_🎬_剧本转换.py
│   ├── 5_💬_AI聊天.py
│   └── 6_⚙️_系统设置.py
├── utils/              # 工具函数
│   ├── __init__.py
│   ├── api.py          # API 调用封装
│   └── config.py       # 配置管理
├── requirements.txt    # 依赖文件
└── README.md
```

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 启动后端 API

```bash
# 在项目根目录
python start.py
```

### 3. 启动前端

```bash
# 在 frontend 目录
streamlit run app.py
```

### 4. 访问应用

打开浏览器访问：http://localhost:8501

## 📖 功能说明

### 🏠 首页
- 平台介绍
- 快速开始指南
- 系统状态监控

### ✍️ 小说创作
- 创建新小说
- 生成故事蓝图
- 生成章节大纲
- 生成章节内容

### 📚 小说管理
- 查看所有小说列表
- 查看小说详情
- 删除小说

### 🎬 剧本转换
- 将小说转换为剧本格式
- 支持多种剧本风格

### 💬 AI 聊天
- 与 AI 进行对话
- 切换不同模型
- 流式输出支持

### ⚙️ 系统设置
- 配置后端 API 地址
- 查看可用模型列表
- 健康检查

## 🔧 配置

在 `utils/config.py` 中可以修改：
- API 基础地址
- 默认模型
- 其他配置项

## 📝 注意事项

1. 确保后端 API 已启动（默认端口 8000）
2. 首次使用请在"系统设置"页面检查连接状态
3. 所有数据存储在后端，前端仅负责展示和交互
